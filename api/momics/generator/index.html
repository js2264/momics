class momics.generator.RangeGenerator(momics, ranges, batch_size, data, label, label_center=None)[source]

This class is implemented to train deep learning models, where the input data is a track or a sequence and the label is another track. The generator will iterate over the ranges in batches and extract the data and label for each range.
