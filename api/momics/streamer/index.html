MomicsStreamer

This class is implemented to efficiently query a momics repository by batches

Module Contents

class momics.streamer.MomicsStreamer(momics, ranges, batch_size=None, features=None, preprocess_func=None, silent=True)[source]

This class is implemented to efficiently query a momics repository by batches and extract any coverage data from it. The data streamer will iterate over ranges in batches and iteratively query a momics.

For a tensorflow DataSet constructor, see momics.dataset.MomicsDataset.
